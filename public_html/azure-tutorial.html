Azure Cloud Tutorial

What is Azure Cloud?

Azure Cloud is a cloud service that allows you to build, run, and manage applications across devices. In this case, Azure Cloud will be hosting the website you are creating.

Setup

The only thing you have to do is create an Azure account. The main tutorial will go over the necessary steps needed to deploy your website to the cloud.
