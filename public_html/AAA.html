AAA Testing Tutorial

What is AAA Testing?

AAA stands for Arrange-Act-Assert. It refers to the pattern that test methods should follow.
1.) Arrange inputs and operations. Setup any required objects, variables, data, or operations for the test to work.
2.) Act on the target behavior. Run the thing that is to be tested (like a method).
3.) Assert expected results. Write an assert statement that says "the result should be: " and verify it's the same as the actual answer.

Example

The image demonstrates a simple example of AAA testing.
The function func takes a parameter named value, adds 1 to it, and returns the new value. The function func_test tests the func function by:
1.) Arranging a variable, value = 1, that will need to be passed into func
2.) Acting on the function by calling func(value)
3.) Asserting that the result is the expected value of 2 (from 1 + 1)

The second test function: test_func_combined shows a more concise version of AAA testing, by combining all 3 steps into one line.
